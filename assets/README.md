# CloudBrew Assets

## Required Images (WebP format)

Place the following optimized WebP images in this folder:

1. **hero.webp** - Hero section coffee image (800x800px recommended)
2. **cappuccino.webp** - Cloud Cappuccino menu item (600x400px recommended)
3. **espresso.webp** - AI Espresso Shot menu item (600x400px recommended)
4. **latte.webp** - Latte Neural Blend menu item (600x400px recommended)

## Image Optimization Tips

- Use WebP format for better compression and quality
- Recommended tools:
  - Online: https://squoosh.app/
  - CLI: `cwebp` (from Google WebP tools)
  - Photoshop: Export as WebP

## Download Source Images

You can download coffee images from:
- Unsplash: https://unsplash.com/s/photos/coffee
- Pexels: https://www.pexels.com/search/coffee/

Then convert to WebP and place in this folder.
